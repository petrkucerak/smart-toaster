# Smart toaster

The goal of this project is to create own server for toaster control built on the ESP8266MOD chip.

## Chip details

(WeMos D1 mini V1)

- Microcontroller: ESP8266
- Frequency: 80MHz
- Flash: 4MB
- RAM: 80KB
- Vendor: WEMOS (fake -> DoIt)
